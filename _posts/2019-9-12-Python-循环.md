循环条件必须是可迭代对象。

| 语法                  | 作用               |
| :-------------------- | :----------------- |
| range(起始,结束,步长) |                    |
| for i in 条件:        | 有限循环和循环遍历 |
| while 条件:           | 无限循环           |
| continue              | 结束当次循环       |
| break                 | 跳出循环           |
