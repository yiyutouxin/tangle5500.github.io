# 多行合并一行

```
""" 正则表达式 """

\r # 查找目标
   # 替换为
```

# 链接

```
""" 普通 """"

http  # 查找目标
<http # 替换为

$     # 查找目标
>     # 替换为
```

# 删除空行

```
""" 正则表达式 """

\r\n\r\n # 查找目标
         # 替换为
```

# 去除多余空格

宏 — 去除多余空格并保存

# 删除奇数行

```
""" 正则表达式 """

^[^\n]*\n([^\n]*) # 查找目标
\1                # 替换为
```
