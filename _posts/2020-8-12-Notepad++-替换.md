# 多行合并一行

```
# 正则表达式

\r # 查找目标
   # 替换为
```

# 链接

```
# 普通

http  # 查找目标
<http # 替换为

$     # 查找目标
>     # 替换为
```

# 删除空行

```
# 正则表达式

\r\n\r\n # 查找目标
         # 替换为
```
