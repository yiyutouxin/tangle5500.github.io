# 初始化

- Ctrl + M

- 格式：H.264

- 输出名称

- 比特率编码：VBR，一次

- 保存预设

# 720p

- 目标比特率：2.5

- 最大比特率：5

# 1080p

## 视频

- 宽度：1920

- 高度：1080

- 帧数率：23.976

- 目标比特率：5.9

- 最大比特率：11.8

- 勾选**关键帧距离**

## 音频

- 采样率：44100 Hz

- 声道：立体声

# 1080p+

## 视频

- 宽度：1920

- 高度：1080

- 帧数率：60

- 目标比特率：5.9

- 最大比特率：11.8

- 勾选**关键帧距离**

## 音频

- 采样率：44100 Hz

- 声道：立体声

# 4K

## 视频

- 宽度：3840

- 高度：2160

- 帧数率：23.976

- 目标比特率：19

- 最大比特率：38

- 勾选**关键帧距离**

## 音频

- 采样率：44100 Hz

- 声道：立体声
