# 添加用户

```
// 创建新用户user1
sudo useradd -m -s /bin/bash user1
// 设置新添加用户的密码
sudo passwd user1
```

# 切换用户

```
// 注销当前用户
注销
// 切换 root 用户，并加载环境配置
su -
// 设置管理员密码
sudo passwd root
// 切换 user 并加载 user 的环境配置，ls -a bash* 查看环境配置
su - user
```

# 修改用户

```
// 把user1用户名修改成666
sudo usermod -l 666 user1
// 添加用户描述信息，id x 查看
sudo usermod -c "用户描述信息"
// 添加次要组
usermod -G x,adm,cdrom
// 修改当前用户名的密码
passwd
// 查看id内的组，uid
id x
// 添加组
groupadd group1
// 指定grd
sudo groupadd -g 2000 group1
// 删除组
groupdel group1
```

# 删除用户

```
// 只删除用户不删除用户创建的文件
suerdel user1
// 删除用户和用户创建的所有文件信息
suerdel -r user1
// 强制删除用户和用户创建的所有文件信息(用户正在使用的时候)
suerdel -f user1
```
