# Ubuntu

0. 浏览 <https://nodejs.org/zh-cn/download/>
0. 下载 `Linux 二进制文件 (x64)`
0. 命令
    ```
    sudo tar -xJvf node-v14.15.1-linux-x64.tar.xz -C /usr/local/lib/nodejs/
    ```
0. 环境变量
    ```
    export PATH=/usr/local/lib/nodejs/node-v14.15.1-linux-x64/bin/:$PATH
    ```
