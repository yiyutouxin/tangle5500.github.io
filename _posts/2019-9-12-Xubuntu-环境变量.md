# 环境变量

| 命令                              | 作用                   |
| --------------------------------- | ---------------------- |
| export                            | 查看所有环境变量       |
| echo \$PATH                       | 查看 PATH 环境变量     |
| export PATH=/home/x/windows:$PATH | 添加 PATH 临时环境变量 |

## 用户环境变量

```
vim ~/.bashrc
export PATH="/home/x/windows:$PATH"
```

## 系统环境变量

```
vim /etc/profile
export PATH="/home/x/windows:$PATH"
source /etc/profile
```
