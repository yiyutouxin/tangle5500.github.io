| -      | 减法 |
| *      | 乘法 |
| /      | 除法 |
| %      | 取余 |

# 比较运算符

| 运算符 | 作用                                               |
| ------ | -------------------------------------------------- |
| ==     | 等于                                               |
| ===    | 绝对等于，值和类型均相等。                         |
| !=     | 不等于                                             |
| !==    | 不绝对等于，值和类型有一个不相等，或两个都不相等。 |
| >      | 大于                                               |
| <      | 小于                                               |
| >=     | 大于等于                                           |
| <=     | 小于等于                                           |
