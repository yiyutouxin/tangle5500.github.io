# 插件

打开 notepad++ 插件 — plugins Admin — 选择 Available — 勾选 NppExec — 点击 Install — 确定立即重启 插件就安装好了

| 插件     | 作用        |
| -------- | ----------- |
| NppExec  | C# 编译环境 |
| Explorer | 目录树      |

# 替换

把多行换成一行

- 选择正则表达式
- 空换成 \n 即可
