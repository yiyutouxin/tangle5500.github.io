```shell
npm config set registry https://registry.npm.taobao.org        设置镜像源
npm config get registry                                        验证镜像源
npm install -g cnpm --registry=https://registry.npm.taobao.org 安装 cnpm
cnpm install xxx                                               安装扩展包
```
