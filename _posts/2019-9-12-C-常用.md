| 语法                | 作用                                                         |
| ------------------- | ------------------------------------------------------------ |
| # include <stdio.h> | 是预处理器指令，告诉 C 编译器在实际编译之前要包含 stdio.h 文件。 |
| int main()          | 是主函数，程序从这里开始执行。                               |
| return 0;           | 终止 main() 函数，并返回值 0。                               |

``` c
#include <stdio.h> 
int main() 
{
printf("Hello, World! \n"); 
return 0; 
} 
```
