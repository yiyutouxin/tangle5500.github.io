| 语法     | 作用                 |
| -------- | -------------------- |
| try:     | 执行此代码           |
| except:  | 发生异常执行此代码   |
| else:    | 没有异常执行此代码   |
| finally: | 是否异常都执行此代码 |

```python
try:
    int('demo')
except(NameError,ValueError):
    print('demo1')
except:
    print('demo2')
else:
    print('demo3')
finally:
    print('demo4')
```

# 捕获异常

| 语法                    | 作用         |
| ----------------------- | ------------ |
| except ValueError as e: | 捕获指定异常 |
| except Exception as e:  | 捕获所有异常 |

``` python
try:
    int('demo')
except NameError as e:
    print('NameError',e)
except ValueError as e:
    print('ValueError',e)
except Exception as e:
    print('Exception',e)
    
    import time
    w = open('log.txt','a+')
    w.write('%s     %s\n'%(time.ctime(),e))
    w.close
```

# 抛出异常

| 语法                     | 作用     |
| ------------------------ | -------- |
| raise Exception('Error') | 抛出异常 |

````python
try:
    raise Exception('Error')
except Exception as e:
    print(e)
````

# 异常判断

| 语法          | 作用                     |
| ------------- | ------------------------ |
| assert 1 == 2 | 条件成立继续执行否则报错 |

```python
assert 1 == 2
```
