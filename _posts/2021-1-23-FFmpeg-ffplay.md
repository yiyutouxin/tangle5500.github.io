# 选项

```
-x 1920      # 强制宽度
-y 1080      # 强制高度
-ss 00:00:10 # 指定位置
-an          # 禁止音频
-vn          # 禁止视频

-f mp4       # 强制格式
-loop 0      # 循环播放
```

# 快捷键

```
f         # 全屏
s         # 下一帧
左        # 快进 10秒钟
上        # 快进 1分钟
Page Down # 快进 10分钟

w  # 切换模式
```
