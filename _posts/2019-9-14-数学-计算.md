# 倒数

除以一个数等于乘以它的倒数

|                    | 倒数            |
| ------------------ | --------------- |
| $\frac{4}{3}$      | $\frac{3}{4}$   |
| $-12$              | $-\frac{1}{12}$ |
| $0.25=\frac{1}{4}$ | $4$             |

# 分数

## 整数乘以分数

$2\times\frac{3}{4}=\frac{2\times3}{4}$

## 约分

- 把分数化成最简分数的过程就叫约分
- 算式中分数和分数约分必须是乘法
- 约分负负得正
