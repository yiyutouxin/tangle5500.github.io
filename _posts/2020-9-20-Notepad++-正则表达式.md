| 转义 | 正类           | 作用 | 负转义 | 负类            | 作用   |
| ---- | -------------- | ---- | ------ | --------------- | ------ |
| `\d` | `[0-9]`        | 数字 | `\D`   | `[^0-9]`        | 非数字 |
| `\l` |                | 小写 | `\L`   |                 | 非小写 |
| `\s` | `[\t]`         | 空白 | `\S`   |                 | 非空白 |
| `\w` | `[A-Za-z0-9_]` | 单词 | `\W`   | `[^A-Za-z0-9_]` | 非单词 |

```
[^A-Za-z] # 除字母字符之外的任何一个字符
{3}       # 指定前面的字符或分组的出现次数
()        # 影响表达式匹配顺序
```
