# 函数

## 随机数

1. 随机：0.85 至 0.9
2. 小数位数：2

```
=ROUND(0.85+(0.05)*RAND(),4)
```

## IF

公式错误显示 None 否则正常显示

```
=IF(ISERROR(VLOOKUP($A11,字典!$A:$AD,COLUMN(),0)),"None",(VLOOKUP($A11,字典!$A:$AD,COLUMN(),0)))
```

# 自定义函数

1、进入 vba 编辑器

2、插入模块

3、在模块中写入如下代码

```vb
Function demo(x, y As Integer)
demo = x * y
End Function
```

| 数值型   | 作用         |

| Single   | 单精度浮点型 |
| Double   | 双精度浮点型 |
