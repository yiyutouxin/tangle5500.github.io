# 原音

元音有时又称作母音，元音总共有 24个。元音可以分为单元音和双元音，而根据元音的不同又可以分为长元音和短元音。所有的元音发声都需要张嘴振动声带。几乎所有单词都含有元音。因而我们可以说元音是构成英语单词发音的基本元素。

## 单元音

| 单元音 | 读音 | 元音   | 注意                                                         |
| ------ | ---- | ------ | ------------------------------------------------------------ |
| [i]    | 衣   | 长元音 | 嘴角微微张开，牙齿分开，舌尖稍微抵住下面的牙齿，并且保持嘴角尽量向两边移，振动声带，但要拉的更长。 |
| [I]    | 衣也 | 短元音 | 嘴唇和牙齿要比发 [i] 时微开，双唇扁平，嘴角两旁肌肉要比发 [i] 时略为放松，舌尖稍微抵住下齿，振动声带。 |
| [?]    | aì   |        | 嘴巴全张，下巴尽量往下拉。舌尖抵住下齿，振动声带。           |
| [ε]    | 也   |        | 嘴巴要比发 [I] 时微开。嘴唇与牙齿再张一些，舌头平放，舌尖轻轻抵住下齿，脸部肌肉放松，振动声带。故意拉长这个音，尾音的部分就像 [e] 的发音。 |
| [ɑ]    | à    |        | 嘴巴要全张。同时舌头自让平放，舌尖不接触下齿，也不要卷起或翘起，振动声带。 |
| [?]    | 饿   |        | 声音要强而且短促，嘴唇与牙齿微开，脸部肌肉放松，舌尖轻抵下齿龈，振动声带。 |
| [?]    | 饿   |        | [?] 的发音要比 [?] 轻，且多只出现在多音节词中的非重读音节部分，而[?] 则出现在单音节词或者是多音节词中的重读部分。 |
| [u]    | 乌   | 长元音 | 嘴形稍扁，嘴唇微微撅起，只留一个小气孔，牙齿微张，声带部分肌肉略为紧张，振动声带。 |
| [?]    | 饿   | 短元音 | [?] 与 [u] 的发音原理大致相同，但是嘴形要更扁。将嘴唇稍稍撅起，形成的气孔要比发 [u] 时稍大。牙齿微张，声带部分肌肉略为紧张，振动声带。 |
| [?]    | o    |        | 嘴唇和牙齿张开，舌头自然垂放，振动声带。                     |

# 辅音

辅音（字音）包括浊辅音（震振声带）和清辅音（不振动声带）。

## 清辅音

| 清辅音 | 元音前 | 元音后 | 字母          | 注意                                                         |
| ------ | ------ | ------ | ------------- | ------------------------------------------------------------ |
| [p]    | po     | pu     |               | 先闭合嘴唇，然后压迫气息使之突破嘴唇而出，不振动声带。       |
| [t]    | te     |        |               | [t] 的发音与汉语拼音 t 的无声音相同。发此音时，先将嘴唇微张，舌尖抵住上齿龈，憋气，然后将舌尖弹开，不振动声带。 |
| [k]    | ke     |        |               | 此音标类似于汉语拼音 k 的无声音。不振动声带。                |
| [f]    | fu     |        | f             | 上齿轻咬下嘴唇内侧，不振动声带。                             |
| [θ]    |        |        | th            | 舌尖微微伸出，置于上下门齿之间，舌身成扁平。气流由舌齿间泻出，摩擦成音，舌尖和下齿之间的气流通道非常窄小，气流从舌齿间的窄缝中泄出。 |
| [s]    |        |        | s \| ss \| ce | 嘴唇微张，牙齿闭合，不振动声带。                             |
| [?]    | shi    |        | sh            | 双唇翘起，牙齿闭合，舌头轻翘，向外吐气。                     |
| [t?]   | qi     |        | ch            | 双唇翘起，牙齿微张，向外吐气。                               |
| [h]    | he     |        | h             | 嘴半张，上下齿张开，舌头平放，向外呵气。                     |
