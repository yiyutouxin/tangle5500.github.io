Python 默认编码为 Unicode。

| 语法                     | 作用                  |
| ------------------------ | --------------------- |
| # coding=utf-8           | 声明文件为 utf-8 编码 |
| # -\*- coding: utf-8 -*- | 声明文件为 utf-8 编码 |

# 编码类型

| 编码类型 |
| -------- |
| Unicode  |
| UTF-8    |
| ASCII    |
| GBK      |
| GB2312   |

# 编码转换

| 编码                 | 作用             | 解码                 | 作用             |
| -------------------- | ---------------- | -------------------- | ---------------- |
| 字符串.encode('GBK') | Unicode 转换 GBK | 字符串.decode('GBK') | GBK 转换 Unicode |

# 类型转换

## bytes 转 float

```python
a = b'123.456'
b = a.decode('utf-8')
c = float(b)
```
