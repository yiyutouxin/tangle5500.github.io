# if 语句

```javascript
if (1)
{
    age = 233;
}
else if (2)
{
    name = "easy"
}
else
{
    age = 666;
}
```

# switch 语句

```javascript
switch(变量)
{
    case 1:
        alert(233);
        break;
    case 2:
        alert(666);
        break;
    default:
        alert(213);
}
```
