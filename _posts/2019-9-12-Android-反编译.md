# 准备工作

0. APK查壳工具_1.5.3_sign.apk
0. java 环境
0. apktool助手.apk
0. dex2jar
0. jd-gui

# 查看源码

0. 将 apk 用 apktool助手反编译dex 得到 calsses.dex 文件
0. 将 classes.dex 反编译成 Java 源码，执行 d2j-dex2jar.bat classes.dex 得到 classes-dex2jar 文件
0. 用 jd-gui 打开 classes-dex2jar 文件
