# 格式转换

```
import ffmpy
ff = ffmpy.FFmpeg(
    inputs={'in.mp4': None},
    outputs={'out.avi': None}
)
print(ff.cmd) # ffmpeg -i in.mp4 out.avi
ff.run()
```

## 批量

```
-vcodec
-acodec
-f
```

```
for %%a in (".\*.mp4") do ffmpeg -i "%%a" -vcodec copy -acodec copy -f mpegts ".\%%~na.mkv"
pause
```
