# 格式转换

```
import ffmpy
ff = ffmpy.FFmpeg(
    inputs={'in.mp4': None},
    outputs={'out.avi': None}
)
print(ff.cmd) # ffmpeg -i in.mp4 out.avi
ff.run()
```

## 批量格式转换

```
for %%a in (".\*.mp4") do ffmpeg -i "%%a" -vcodec copy -acodec copy -f mpegts ".\%%~na.mkv"
pause
```

# 转码

```
-c:a
-c:v
```

```
import ffmpy
ff = ffmpy.FFmpeg(
    inputs={'in.mp4': None},
    outputs={'out.mp4': '-c:a mp2 -c:v mpeg2video'}
)
print(ff.cmd) # ffmpeg -i in.mp4 -c:a mp2 -c:v mpeg2video out.mp4
ff.run()
```
