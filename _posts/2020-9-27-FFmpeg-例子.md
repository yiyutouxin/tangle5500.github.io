# 格式转换

```
import ffmpy
ff = ffmpy.FFmpeg(
    inputs={'in.mp4': None},
    outputs={'out.avi': None}
)
print(ff.cmd) # ffmpeg -i in.mp4 out.avi
ff.run()
```

## 批量

```
-codec copy # 表示所有的流都不进行再次编码
```

```
set in=mkv
set out=mp4
for %%a in (".\*.%in%") do ffmpeg -i "%%a" -codec copy ".\%%~na.%out%"
pause
```
