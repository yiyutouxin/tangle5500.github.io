# 时间函数

可以写在表中字段的默认值中

| 语法   | 作用                   |
| ------ | ---------------------- |
| Date() | 返回当前系统日期       |
| Time() | 返回当前系统时间       |
| Now()  | 返回当前系统日期和时间 |
