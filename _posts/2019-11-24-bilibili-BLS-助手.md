# 项目地址

https://github.com/mscststs/BLS

# 使用说明

1. 安装 https://github.com/mscststs/BLS/releases
1. 安装 node
1. 安装 yarn

```
npm install yarn
```
1. 运行

```
yarn install 安装依赖
yarn run dev 运行程序
```
