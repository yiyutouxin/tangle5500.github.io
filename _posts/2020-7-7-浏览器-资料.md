# Chrome

- 浏览 `chrome://version`

- 个人资料路径

```
%USERPROFILE%\AppData\Local\Google\Chrome\User Data\Default            # 个人资料路径
%USERPROFILE%\AppData\Local\Google\Chrome\User Data\Default\Extensions # 扩展程序
```
