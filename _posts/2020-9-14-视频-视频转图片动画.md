# Xubuntu

0. Xubuntu x64
    ```
    sudo apt install git
    git clone https://github.com/HypoX64/bilibili.git
    cd bilibili/OneImage2Video
    sudo apt install python3-opencv
    sudo apt install ffmpeg
    ```
0. 在 main.py 中修改视频和图片路径
    ```
    python3 main.py
    ```

# Windows

0. Windows 10
    ```
    pip3 install opencv-python
    ```
