| 语法                          | 作用                           |
| ----------------------------- | ------------------------------ |
| 变量.isalnum()                | 所有字符都是数字或者字母       |
| 变量.isalpha()                | 所有字符都是字母               |
| 变量.isdigit()                | 所有字符都是数字               |
| 变量.islower()                | 所有字符都是小写               |
| 变量.isupper()                | 所有字符都是大写               |
| 变量.istitle()                | 所有单词都是首字母大写，像标题 |
| 变量.isspace()                | 所有字符都是空白字符 \t \n \r  |
| type(eval('123')) == int      | 判断字符串是否为 int           |
| type(eval("123.23")) == float | 判断字符串是否为 float         |
