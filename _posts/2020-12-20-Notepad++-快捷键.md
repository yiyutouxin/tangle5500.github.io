# 文件

```
ctrl + o   # 打开
ctrl + tab # 标签切换
ctrl + w   # 关闭当前标签
```

# 编辑

```
alt + shift + 方向键 # 列编辑
alt + 鼠标           # 列编辑
alt + c              # 列编辑
ctrl + t             # 当前行与上一行交换位置
ctrl + l             # 删除当前行
ctrl + g             # 行跳转
ctrl + backspace     # 向后删除到空格的字符
ctrl + delete        # 向前删除到空格的字符
ctrl + b             # 配对括号跳转
```

# 搜索

```
ctrl + h         # 替换
f3               # 搜索下一个结果
shift + f3       # 搜索上一个结果
ctrl + shift + f # 文件查找
```

# 运行

```
f5                     # 运行
ctrl + alt + shift + r # 在 chrome 中打开
```
