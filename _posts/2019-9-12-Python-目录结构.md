# 调用函数

a.py

```python
def _def():
    print(233)
```

b.py

```python
import a
b._def()
```

b.py

```python
form a import _def
_def()
```

# 调用类

a.py

```python
class _class:
    def __init__(self,xx):
        self.x = xx
    def _def(self):
        print(self.x)
```

b.py

```python
from a import _class
a = (233)
a._def()
```

b.py

```python
import a
a = a._class(233)
a._def()
```

## 在不同文件夹下

a.py 文件路径 C:

b.py

```python
import sys
sys.path.append(r'c:') # 将 a.py 路径加入到 sys.path 路径中
```
