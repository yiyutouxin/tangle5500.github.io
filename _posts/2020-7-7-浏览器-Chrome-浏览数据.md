# 浏览数据路径

1. 浏览器输入：chrome://version
1. **个人资料路径**就是浏览数据路径
    ```
    \Chrome\User Data\Default
    C:\Users\Tangle\AppData\Local\Google\Chrome\User Data\Default
    ```

## 扩展程序

```
\Chrome\User Data\Default\Extensions
```
