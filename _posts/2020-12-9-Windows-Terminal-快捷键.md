# 控制台

<https://docs.microsoft.com/zh-cn/windows/terminal/panes>

```
ctrl + alt + num     # 切换控制台
shift + alt + =      # 拆分新垂直窗格
shift + alt + -      # 拆分新水平窗格
alt + 箭头键         # 切换窗格
shift + alt + 箭头键 # 窗格尺寸
alt + shift + d      # 复制窗格
ctrl + shift + w     # 关闭当前控制台
```
