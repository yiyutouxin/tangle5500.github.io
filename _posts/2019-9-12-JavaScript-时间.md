| 语法                       | 作用          |
| -------------------------- | ------------- |
| Date();                    | 中国标准时间  |
| Date().getHours();         | 小时（0-23）  |
| Date().getDay();           | 周（0-6）     |
| Date().getDate();          | 日（1-31）    |
| Date().getFullYear() ;     | 年份（2019）  |
| Date().getMilliseconds() ; | 毫秒（0-999） |
| Date().getMinutes() ;      | 分钟（0-59）  |
| Date(). getMonth() ;       | 月份（0-11）  |
| Date(). getSeconds() ;     | 秒（0-59）    |
| Date(). getTime() ;        | 时间戳毫秒    |
