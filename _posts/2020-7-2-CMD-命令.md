# 常用

| 语法                   | 作用                                 |
| ---------------------- | ------------------------------------ |
| tree /f                | 目录树                               |
| 命令 /?                | 查看帮助                             |
| systeminfo             | 查看处理器位数                       |
| start demo.txt & exit  | & 前一条命令执行无论如何都执行后一条 |
| start demo.txt && exit | & 前一条命令执行成功执行后一条       |

# taskkill

| taskkill 参数 | 作用                   |
| ------------- | ---------------------- |
| /im           | 进程名称               |
| /f            | 指定强制终止进程       |
| /pid          | 指定要终止的进程的 PID |

```shell
taskkill /f /im cmd.exe
```

# tree

| tree 参数 | 作用               |
| --------- | ------------------ |
| /f        | 包含子目录的目录树 |

# 删除

| rd 参数 | 作用                 |
| ------- | -------------------- |
| /q      | 无需确认             |
| /s      | 删除自身和所有子目录 |

| 命令 | 作用       |
| ---- | ---------- |
| rd   | 删除空目录 |
| del  | 删除文件   |

# start

| 命令        | 作用           |
| ----------- | -------------- |
| start 1.txt | 打开文件或目录 |

# 循环

| 命令                         | 作用 |
| ---------------------------- | ---- |
| for %i in (1,2,3) do echo %i | 遍历 |

# 路径

| cd 参数 | 作用         |
| ------- | ------------ |
| /d      | 切换指定目录 |

| 命令     | 作用            |
| -------- | --------------- |
| cd       | 切换目录        |
| %windir% | 代表 C:\Windows |

# echo

| 语法                     | 作用       |
| ------------------------ | ---------- |
| echo 内容>文件名.后缀名  | 覆盖写入。 |
| echo 内容>>文件名.后缀名 | 追加写入。 |
