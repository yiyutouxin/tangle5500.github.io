# Emmet

## 安装

浏览 <https://github.com/emmetio/npp#readme>

## 快捷键

运行 — 管理快捷键 — Plugin commands — Plugin — EmmetNPP.dll — Expand abbreviation

```
ctrl + e
```

## 自定义

- `Notepad++\plugins\EmmetNPP\emmet\snippets.json`

```
"html": {
    "filters": "html",
    "profile": "html",
    "snippets": {
        "new": "new",
    },
```
