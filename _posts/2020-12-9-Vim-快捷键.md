# 快捷键

## 折叠

```
:help fold-marker
```

```
space # 当前行展开关闭折叠
zM    # 关闭所有折叠
zR    # 展开所有折叠
zi    # 翻转折叠
```

# 映射键

```
:map  # 普通、可视、选择、操作符等待
:nmap # 普通
:vmap # 可视与选择
:smap # 选择
:xmap # 可视
:omap # 操作符等待
:map! # 插入与命令行
:imap # 插入
:lmap # 插入、命令行、Lang-Arg
:cmap # 命令行
:tmap # 终端作业
```

## 缩进

```
nmap <tab> V>
nmap <S-tab> V<
vmap <tab> >gv
vmap <S-tab> <gv
```
