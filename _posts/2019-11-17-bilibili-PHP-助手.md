# 项目地址

https://github.com/lkeme/BiliHelper-personal

## 使用指南

### 克隆项目

```shell
git clone https://github.com/lkeme/BiliHelper-personal.git
cd BiliHelper-personal/conf
cp user.conf.example user.conf
```

### 安装依赖

```shell
composer install --ignore-platform-reqs
```

### 配置文件

```text
/BiliHelper/conf
```

### 执行程序

```shell
php index.php
```
