# Shell

```
set s = CreateObject("wscript.shell")
s.run run_main, 0
```

# File

```
set f = createobject("scripting.filesystemobject")
file_dir = f.GetFile(Wscript.ScriptFullName).ParentFolder.Path ' 获取文件目录
ws.currentdirectory = file_dir                                 ' 切换工作路径
```
