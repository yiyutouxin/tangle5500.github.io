# 映射键

## 模式关系

```
:map  # 普通、可视、选择、操作符等待
:nmap # 普通
:vmap # 可视与选择
:smap # 选择
:xmap # 可视
:omap # 操作符等待
:map! # 插入与命令行
:imap # 插入
:lmap # 插入、命令行、Lang-Arg
:cmap # 命令行
:tmap # 终端作业
```

## 延时

```
set timeoutlen=0  " 映射键
set ttimeoutlen=0 " 模式切换
```

## Alt

```
execute "set <M-j>=\ej"
nmap <M-j> :echo "hello world"<CR>
```

# 缩进

```
" 缩进
    nmap <tab> V>
    nmap <S-tab> V<
    vmap <tab> >gv
    vmap <S-tab> <gv
```

# 拆分窗格

```
" 拆分窗格
    nmap s= :resize +3<CR>          " 水平
    nmap s- :resize -3<CR>          " 水平
    nmap v= :vertical resize +3<CR> " 垂直
    nmap v- :vertical resize -3<CR> " 垂直
```
