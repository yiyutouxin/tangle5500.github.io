# monokai

## Ubuntu

0. <https://github.com/sickill/vim-monokai>
0. 安装
    ```
    cd ~/.vim/bundle
    git clone https://github.com/sickill/vim-monokai.git
    cp -r ~/.vim/bundle/vim-monokai/colors/ ~/.vim
    ```
0. `.vimrc`
    ```
    " 代码高亮
    syntax enable
    colorscheme monokai
    ```

## Windows

0. 打开目录 `gvim_8.2.1724_x86\vimfiles\bundle`
0. 安装
    ```
    git clone https://github.com/sickill/vim-monokai.git
    copy vim-monokai\colors ..\colors
    ```
0. `_vimrc`
    ```
    " 代码高亮
    syntax enable
    colorscheme monokai
    ```
