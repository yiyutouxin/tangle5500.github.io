# 替换

```
# 替换标志

g # 全局替换
i # 大小写不敏感
I # 大小写敏感
c # 确认
e # 不显示错误
```

```
:s/1/2/g      # 当前行 1 替换成 2，g 为所有出现都会被替换
:%s/1/2/g     # 全文 1 替换成 2，g 为所有出现都会被替换
:'<,'>s/1/2/g # 选区 1 替换成 2，g 为所有出现都会被替换
:.,+2s/1/2/g  # 当前行至 +2 行 1 替换成 2，g 为所有出现都会被替换
```

# 正则表达式

```
.      # 任何字符
^      # 行首
$      # 行尾

*      # 0 次或多次
\+     # 1 次或多次
\?     # 0 次或 1 次

[abc]  # 匹配 abc 任意单个字符
[^abc] # 匹配非 abc 任意单个字符
\{0,1} # 指定前面的字符或分组的出现次数
\(\)   # 分组匹配影响表达式匹配顺序
```

# 例子

```
# 文本
https://www.google.com
https://www.baidu.com

# 正则
:.,2s/\(https\?:\/\/\)\(www\.*\.com\)/\2 # 替换当前行至第 2 行

# 结果
www.google.com
www.baidu.com
```
