```
.      # 任何字符
^      # 行首
$      # 行尾

*      # 0 次或多次
\+     # 1 次或多次
\?     # 0 次或 1 次

[abc]  # 匹配 abc 任意单个字符
[^abc] # 匹配非 abc 任意单个字符
\{0,1} # 指定前面的字符或分组的出现次数
\(\)   # 分组匹配影响表达式匹配顺序
```

# 例子

```
# 文本
https://www.google.com
https://www.baidu.com

# 正则
:.,2s/\(https\?:\/\/\)\(www\.*\.com\)/\2 # 替换当前行至第 2 行

# 结果
www.google.com
www.baidu.com
```
